A1: Blink!

5 LEDS blinking in consecutive pairs


SCHEMATIC

This schematic shows 5 different colored LEDS lined up in a parallel circuit. The Arduino is powered by a 5V USB cord where 5 different pins are used to individually connect to each LED. Depending on the color of the LED, resistors with specific values were calculated and selected (see "Calculating Resistors" below) to avoid damaging components on the circuit. All the resistors were then connected to ground.


CIRCUIT

Following the schematic outlined above, I built the circuit using the following components:

  • 1 Arduino Uno R3

  • 1 breadboard

  • 1 USB cable and power source (laptop)

  • 6 wires (1 for ground; 5 for the LEDs)

  • 5 LEDs (1 each of green, red, blue, yellow, and white)

  • 5 resistors (3 of 220Ω; 2 of 100Ω)

CALCULATING RESISTORS


For Green, Red, and Yellow LEDs — knowing that they have a 1.8 voltage drop and that the USB provides 5V, I calculated the voltage across the resistor as follows:

V = 5V - 1.8V = 3.2V

Using Ohm's Law, I then calculated the value of the resistors to achieve the desired current of 0.02A:

voltage = current * resistance

3.2V = 0.02A * resistance

resistance = 3.2V / 0.02A = 160 Ω

Because the class kit provided does not include 160 Ω resistors, I opted for a higher value of 220 Ω which resulted in a lesser current but would prevent the LEDs from breaking.



Using similar calculation for the Blue and White LEDs that have a 3.3 voltage drop, the voltage across the resistor would be

V = 5V - 3.3V = 1.7 V

Using Ohm's Law,

voltage = current * resistance

1.7V = 0.02A * resistance

resistance = 1.7V / 0.02A = 85 Ω

Again, because the kit does not include 85 Ω resistors, I used 100 Ω resistors which resulted in a slightly lesser current than the desired value.

CODE SNIPPET

   
/*this firmware makes 5 LEDs (lined up in a straight line) turn on and off in 
consecutive pairs from one end of the line to another*/ 


// adjustable timer for .2 second (used for delays)
int timer = 200;            


/* initialize each digital pin that corresponds to each LED light
(12 = white, 10 = yellow, 8 = blue, 6 = red, 4 = green) */
void setup() { 
  // use a for loop to select pins 4,6,8,10,12
  for (int thisPin = 4; thisPin <= 12; thisPin += 2) { 
    //set pins as outputs
    pinMode(thisPin, OUTPUT);
  }
}


//infinite loop that makes the 5 LEDs blink in a pattern forever
void loop() { 
  /* use a for loop to initialize the first LED in the pair to light up
  starting with pins 12, 10, 8, 6 respectively */
  for (int firstPin = 12; firstPin >= 6; firstPin -= 2) { 
    /* initialize the second LED in the pair to light up
    whose pin #is 2 less than that of the first */
    int secondPin = firstPin - 2; 
    //turn on the first LED in the pair
    digitalWrite(firstPin, HIGH); 
    //wait for "timer"
    delay(timer); 
    //turn on the second LED in the pair
    digitalWrite(secondPin, HIGH); 
    //wait for "timer"
    delay(timer); 
    //turn off the first LED
    digitalWrite(firstPin, LOW); 
    //turn off the second LED
    digitalWrite(secondPin, LOW); 
    //wait for "timer"
    delay(timer);          
  }
}
