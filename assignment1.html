A1: Blink!

Identifying Values of Resistors
For Green, Red, and Yellow LEDs — knowing that they have a 1.8 voltage drop and that the USB provides 5V, I calculated the voltage as follows:
V = 5V - 1.8V = 3.2V
Using Ohm's Law, I then calculated the value of the resistors to achieve the desired current of 0.02A:
voltage = current * resistance
3.2V = 0.02A * resistance
resistance = 3.2V / 0.02A = 160 Ω
Because the class kit provided does not include 160 Ω resistors, I opted for a higher value of 220 Ω which resulted in 3.2V/220Ω = ~0.0145 A of current which is lower than the desired current but would prevent the LEDs from breaking.
Using similar calculation for the Blue and White LEDs that have a 3.3 voltage drop, the voltage is
V = 5V - 3.3V = 1.7 V
Using Ohm's Law,
voltage = current * resistance
1.7V = 0.02A * resistance
resistance = 1.7V / 0.02A = 85 Ω
Again, because the kit does not include 85 Ω resistors, I used 100 Ω resistors which resulted in a slightly lesser desired current of 1.7V/100Ω = 0.017 A. 

   
/*this firmware makes 5 LEDs (lined up in a straight line) turn on and off in 
consecutive pairs from one end of the line to another*/ 


// adjustable timer for .2 second (used for delays)
int timer = 200;            


/* initialize each digital pin that corresponds to each LED light
(12 = white, 10 = yellow, 8 = blue, 6 = red, 4 = green) */
void setup() { 
  // use a for loop to select pins 4,6,8,10,12
  for (int thisPin = 4; thisPin <= 12; thisPin += 2) { 
    //set pins as outputs
    pinMode(thisPin, OUTPUT);
  }
}


//infinite loop that makes the 5 LEDs blink in a pattern forever
void loop() { 
  /* use a for loop to initialize the first LED in the pair to light up
  starting with pins 12, 10, 8, 6 respectively */
  for (int firstPin = 12; firstPin >= 6; firstPin -= 2) { 
    /* initialize the second LED in the pair to light up
    whose pin #is 2 less than that of the first */
    int secondPin = firstPin - 2; 
    //turn on the first LED in the pair
    digitalWrite(firstPin, HIGH); 
    //wait for "timer"
    delay(timer); 
    //turn on the second LED in the pair
    digitalWrite(secondPin, HIGH); 
    //wait for "timer"
    delay(timer); 
    //turn off the first LED
    digitalWrite(firstPin, LOW); 
    //turn off the second LED
    digitalWrite(secondPin, LOW); 
    //wait for "timer"
    delay(timer);          
  }
}
